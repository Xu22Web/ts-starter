## Description

A unboxed TypeScript project template.

## Usage

```bash
degit https://github.com/xu22web/ts-starter.git ts-starter

cd ts-starter

pnpm install
```

## Development

```bash
# dev
pnpm run dev
```

## Build

```bash
# build
pnpm run build
```

## Test

```bash
# test
pnpm run test
```
